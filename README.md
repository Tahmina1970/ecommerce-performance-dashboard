# E-Commerce Business Intelligence Dashboard

An interactive e-commerce analytics dashboard built with Google Looker Studio to analyze sales performance, conversion metrics, customer behavior, and product category insights.

---

## Dashboard Features

- Revenue tracking
- Conversion rate analysis
- Product category performance
- Sales funnel insights
- Revenue trend visualization
- Channel comparison analysis
- KPI monitoring

---

## Tools Used

- Google Looker Studio
- CSV Dataset
- Data Visualization
- Business Intelligence
- E-Commerce Analytics

---

## Dashboard Pages

### Executive Overview
Includes:
- Revenue
- Sessions
- Purchases
- Conversion Rate
- Add-to-Cart Rate

### Product & Funnel Analytics
Includes:
- Revenue trends
- Product category performance
- Revenue by channel
- Funnel analysis
